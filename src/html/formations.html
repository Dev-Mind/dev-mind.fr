Formation

Dev-Mind propose plusieurs formations sur la programmation web que nous pouvons personnaliser en fonction de vos besoins, de votre contexte, de votre degré de connaissance... Ces formations mettent l'accent sur les travaux pratiques pour vous permettre d'appliquer au plus vite ce que nous apprenons ensemble

Framework Java Spring

Durée : 1 à 7 jours

Le but de cette formation est de comprendre les méandres du framework Spring et d'être capable de créer une application web moderne avec Spring Boot... en savoir plus...

Optimiser une application web

Durée : 1 journée

Le but de cette formation est d'aborder les différentes mécaniques pour améliorer les performances de votre site web (HTTP2, service workers, asynchrone...). Vous apprendrez comment optimiser le loadtime, le firstload... en savoir plus...
