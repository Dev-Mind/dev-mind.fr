Formation Spring

Spring est le framework Java le plus utilisé et il est Open Source. Connaissez vous toute la magie de Spring, comment il fonctionne, comment l'utiliser tout en respectant les bonnes pratiques de développement ?

J'utilise Spring depuis 15 ans et la version 1. Nous verrons ensemble comment le framework a évolué, quels sont les problématiques qui ont été résolues. Ensemble nous pouvons construire une formation à la carte

Je vous propose différents modules en fonction des différents projets et sous projets de Spring.

CORE

  • Injection
  • AOP
  • Web MVC
  • Web Rest
Temps idéal : 3 jours

BOOT

  • Starter
  • Auto config
  • Logs
  • Métriques
Temps idéal : 1 jour

DATA

  • JDBC
  • JPA (Hibernate)
  • Transaction
  • Validation
Temps idéal : 2 jours

SECURITY

  • Authorisation
  • Authentification
  • Cors
  • CSRF
Temps idéal : 1 jour

Tarification

Il existe deux modes de tarification en fonction du mode choisi

  • mode INTRA : la formation vous est réservée et vous pouvez proposer jusqu'à 6 stagiaires
  • mode INTER : formation d'un seul stagiaire

Contactez moi sur guillaume (@) dev-mind.fr pour plus d'informations

La formation est principalement axée sur de la pratique. Si vous faites tous les modules vous construisez de A à Z une application qui permet de lire des données en temps réel et de les afficher sous forme de graphique
